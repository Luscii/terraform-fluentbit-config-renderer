# Architecture Decision Records

Dit directory bevat Architecture Decision Records (ADRs) voor deze Terraform module, opgemaakt volgens het MADR (Markdown Architectural Decision Records) formaat.

## Wat zijn ADRs?

ADRs documenteren significante architecturale beslissingen die gemaakt zijn voor deze module, inclusief:
- De context en het probleem dat opgelost wordt
- Overwogen alternatieven
- De gekozen aanpak met motivering
- Consequenties van de beslissing
- Implementatieplan

## Formaat

Alle ADRs volgen het MADR template zoals gedefinieerd in [adr.instructions.md](../instructions/adr.instructions.md).

## Naamgeving

ADR bestanden gebruiken sequentiële nummering:
- Formaat: `NNNN-titel-met-streepjes.md`
- Nummers zijn zero-padded (4 cijfers)
- Titel is lowercase met streepjes
- Voorbeelden:
  - `0001-use-cloudposse-label-module.md`
  - `0002-implement-auto-scaling.md`
  - `0003-add-service-connect.md`
